# Glucose Prediction using Machine Learning

This project predicts glucose levels using machine learning models and BERT-based embeddings.

## Project Structure
- preprocess.py – Data preprocessing
- train.py – Model training
- main.py – Main execution
- bert_embeds.py – Feature extraction using BERT
- confusion_matrices.py – Model evaluation

## Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas, NumPy
- BERT

## How to Run
1. Create a virtual environment
2. Install dependencies  
   `pip install -r requirements.txt`
3. Train the model  
   `python train.py`
